# Chrome Extension for theoldreader

## 简介
国内好用的RSS web阅读器少之又少，[theoldreader](https://theoldreader.com/)，是我找到的一个比较好用的网站，但是网站的性能太差，我就考虑着写一个浏览器插件

## 官方api文档
https://github.com/theoldreader/api

## 技术栈 

- [模板](https://github.com/JohnBra/vite-web-extension)
- [Vite脚手架](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Ant Design](https://ant.design/)
- [Chrome Extension Manifest Version 3](https://developer.chrome.com/docs/extensions/mv3/intro/)
